# Lexogrine Obserview

Standalone Radar HUD for [LHM.gg](https://lhm.gg) (Lexogrine HUD Manager)
- Integrated Telestrator: Includes built-in drawing tools for tactical analysis (Requires LHM v1.11+).
- Network Synchronization: Real-time canvas syncing allows you to draw on one PC and display it instantly on another.
- Enhanced Visuals: Toggleable player avatars displayed directly on map nodes.

## Keybinds:
### **Left Alt + A**
>Toggles avatars

### **C**
>Clears drawing
